# 目录迁移说明

原有的 `experiments/sensitivity_analysis` 与 `experiments/real_sensitivity_analysis` 已迁移到 `sensitivity_analysis/experiments/archive/`，历史样例，仅供参考。新的训练/推理/评估结果会统一落在 `sensitivity_analysis/{logs,outputs,results}`。如需查看旧实验日志或结果，请转到：

- `sensitivity_analysis/experiments/archive/sensitivity_analysis`
- `sensitivity_analysis/experiments/archive/real_sensitivity_analysis`

请勿在此目录下写入新产物。
